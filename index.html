文本翻译，生成音频

使用说明

获取 Google Text 2 Speech API Key

如果你是 Fd 的朋友，请点击文章获取 API key。该 API key 仅在 Fd 内网条件下可以使用，在家使用请挂 VPN。

如果你想自己申请 API key，请参考文章并遵照操作。

输入申请的 API key ，点击『Save』即可。

翻译文本

1. 在中文内容的文本框中输入需要翻译的中文内容。

2. 点击『翻译成英文』按钮进行翻译。

3. 翻译后的英文内容将会自动填充在英文内容的文本框中。

转写音频

1. 在英文内容的文本框中校正需要转写的英文内容。

2. 从目标转写语言下拉框中选择需要转写的目标语言。

3. 点击『转写音频』按钮进行转写。

4. 转写完成后，对应的 Mp3 文件会自动下载到本地。
